* repo

Clone this repo in your ~/

* spacemacs leader keys
The leader keys are set to:
~M-m --> f19~
~M-m m --> f17~

This way it is possible to assign them to right-cmd and right-option using [karabiner](https://karabiner-elements.pqrs.org/ )
